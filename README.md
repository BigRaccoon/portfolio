# Portfolio System Analyst

Репозиторий содержит учебные проекты, выполненные в рамках изучения системного анализа, моделирования бизнес-процессов и проектирования информационных систем, а также видоизмененные проекты из реального опыта работы


## Используемые инструменты

- PlantUML
- Draw.io
- BPMN 2.0
- Figma
