# Meeting Room Booking System

Учебный проект по системному анализу.

## Описание

Система предназначена для поиска, бронирования и отмены бронирования переговорных комнат. Проект демонстрирует процесс анализа требований, моделирования бизнес-процессов и проектирования модели данных.

## Основные возможности

- Поиск свободных переговорных комнат
- Бронирование комнаты на выбранную дату и время
- Отмена существующего бронирования
- Управление переговорными комнатами администратором
- Уведомление пользователя о результате бронирования
