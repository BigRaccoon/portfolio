LegalSupport CRM

Система управления юридическими обращениями.
Описание проекта

Внутренняя CRM-система для юридической фирмы, автоматизирующая работу с клиентскими обращениями от регистрации до архивации.
Основные возможности

Регистрация обращений с проверкой на дубликаты
Распределение задач между юристами
Делегирование дел между сотрудниками
Завершение работы с подтверждением от клиента
Автоматическое закрытие по тайм-ауту (3 рабочих дня)
Структура проекта

Макет интерфейса/ Прототипы экранов в Figma:
Регистрация нового обращения (Оператор)
Пул задач (Юрист)
Карточка дела (Юрист)
Требования к проекту/
Job Story — сценарий завершения работы по обращению
User Story — функциональные требования (регистрация, взятие в работу, делегирование)
BPMN Процесс... клиента/ Диаграмма бизнес-процесса в нотации BPMN 2.0
Class Diagram/ UML-диаграмма классов системы
ER diagram/ ER-диаграмма базы данных (нотация Crow's Foot)
State Machine Diagram/ Диаграмма состояний обращения (жизненный цикл)
Use case Diagram/ Диаграмма прецедентов (акторы и сценарии использования)
Инструменты

Draw.io — моделирование (UML, BPMN, ER)
Figma — прототипирование интерфейса
