# LegalSupport CRM

Учебный проект по системному анализу.

## Описание

Внутренняя CRM-система для юридической компании, предназначенная для автоматизации обработки клиентских обращений: от регистрации до завершения работы и последующей архивации.

## Основные возможности

* Регистрация новых обращений с проверкой на дубликаты
* Распределение обращений между юристами
* Делегирование дел между сотрудниками
* Подтверждение завершения работы клиентом
* Автоматическое закрытие обращения по тайм-ауту (3 рабочих дня)

## Реализованные артефакты

### Требования

* User Stories
* Job Story
* Функциональные требования

### Диаграммы

* BPMN 2.0 Process Diagram
* Use Case Diagram
* Class Diagram
* ERD (Crow's Foot)
* State Machine Diagram

### Прототипы интерфейса

* Экран регистрации нового обращения
* Пул задач юриста
* Карточка дела
